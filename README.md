# Dokumentasi Proyek Codex

## 🔧 Alat yang Diinstal
- **Cursor IDE** untuk pengembangan dan integrasi AI.  
- **Claude Code (ekstensi di Cursor)** sebagai asisten coding.  
- **Codex (ekstensi di Cursor)** untuk membantu penulisan dan eksekusi kode.  
- **GitHub** sebagai repositori publik untuk menyimpan dan berbagi project.

---

## ✅ Langkah-langkah yang Telah Diselesaikan
1. Membuat akun GitHub dan repositori publik.  
2. Menginstal Cursor IDE di komputer.  
3. Menambahkan ekstensi **Claude Code** dan **Codex** ke Cursor.  
4. Membuat file `README.md` awal di repositori.  
5. Melakukan commit dan push ke GitHub.

---

## ⚠️ Masalah yang Dihadapi & Solusi
**Belum familiar dengan commit message**  
  → Solusi: Memberi pesan commit sederhana seperti "upload file pertama" agar jelas riwayat perubahan.
